# React Native App

##Barcode Scanner
1. Click Cart menu icon.
2. Press Scan.
3. Generate a barcode: https://www.barcodesinc.com/generator/index.php
4. Camera will automatically recognize the barcode and app will show an alert.

##Microsoft Face API

```let api_key = '';```
